Simplifying Algebraic Fractions

An interactive step-by-step exercise on reducing fractions to lowest terms, with formative feedback at every stage.

Please enter your name and class before starting.

Success criteria: I can reduce a given algebraic fraction to its lowest terms.

Work through the tabs in order: Numeric Fractions → Basic Variable Fractions → Fractions, Common and Linear Factors → Fractions, Quadratic Expressions with Common Linear Factors. Each question is broken into sub-steps — enter your answer for each and press Check. If stuck, press Show to reveal the working, then make sure you understand it before continuing.

Worked example (Numeric Fractions):

$$\dfrac{5}{10} = \dfrac{5}{2\times 5} = \dfrac{\cancelto{1}{5}}{2\times\cancelto{1}{5}} = \dfrac{1}{2}$$ — dividing the common factor 5 from both the numerator and denominator.

Worked example (Fractions, Quadratic Expressions with Common Linear Factors):

$$\dfrac{x^2-5x+6}{x^2-4} = \dfrac{(x-2)(x-3)}{(x-2)(x+2)} = \dfrac{\cancelto{1}{(x-2)}(x-3)}{\cancelto{1}{(x-2)}(x+2)} = \dfrac{x-3}{x+2}$$ — factor numerator and denominator fully, then cancel the common linear factor.

Typing tip: use ^ for powers (e.g. x^2) and write fractions as numerator/denominator, e.g. a/b or (x-3)/(x+2). No spaces needed.

Algebraic Fractions
Numeric fractions

Reducing to lowest terms

Find the Highest Common Factor (HCF) of the numerator and denominator, then divide both by it.

Example:

$\dfrac{5}{10}$
$= \dfrac{5}{2 \times 5}$ HCF of 5 and 10 = 5
$= \dfrac{\cancelto{1}{5}}{2 \times \cancelto{1}{5}}$ divide the numerator and denominator by the common factor 5
$= \dfrac{1}{2}$ lowest terms
Variable fractions

Monomials with variables

Simplify the numeric coefficients using the HCF, then simplify each variable by subtracting the smaller exponent from the larger (it stays on the side with the larger power).

Example: $\dfrac{12ab}{a^2} = \dfrac{12b}{a}$

Factorizing Linear Expressions

Factorize first, then cancel

When the numerator and/or denominator are expressions (not single terms), factorize each one completely first. Only then can you cancel a common factor from both.

Example: $\dfrac{2a+6}{2} = \dfrac{2(a+3)}{2} = a+3$ — factorize the numerator first, then divide by the common factor.

Quadratic factoring

Factoring quadratics before cancelling

Factor any quadratic expressions — look out for the difference of two squares ($p^2x^2-q^2y^2=(px-qy)(px+qy)$) and trinomials that factor into two linear brackets. Then cancel the common linear factor.

Example: $\dfrac{4a^2-9b^2}{2a-3b} = \dfrac{(2a-3b)(2a+3b)}{2a-3b} = 2a+3b$
